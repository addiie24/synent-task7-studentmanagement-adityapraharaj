# Student Management System

A menu-driven Python application to manage student records.

## Features

- Add Student
- View Students
- Update Student
- Delete Student
- JSON Data Storage
- Persistent Records

## Technologies Used

- Python
- JSON
- File Handling

## How to Run

```bash
python student_management.py
```

## Example Menu

1. Add Student
2. View Students
3. Update Student
4. Delete Student
5. Exit

## Data Storage

Student records are stored in:

students.json
